#  Chord DHT Simulator

A visual and interactive simulation of the **Chord Distributed Hash Table (DHT)** protocol using **Streamlit**.

This project demonstrates how decentralized systems efficiently locate data using consistent hashing and logarithmic lookup time.

---

## Features

-  Configurable Chord ring (bit size `m`, number of nodes)
- Dynamic node creation
-  Key lookup simulation
-  Performance metrics (hops, latency)
-  Modern redesigned UI (clean, gradient-based theme)

---

##  Concepts Covered

- Distributed Hash Tables (DHT)
- Consistent Hashing
- Finger Tables
- Logarithmic lookup complexity → **O(log N)**
- Peer-to-peer systems
